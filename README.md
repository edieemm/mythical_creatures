Porta Mythical Creatures React App (now featuring, Server & DB!)  

## To Start:  
### create a db called 'mythical_creatures'  

### create tables and data using 'database'.sql' file

### $npm install  

## to run client:  
### $npm run client  
(will be at localhost: 3000)  

## to run server:  
### $npm run server  
(will be at localhost: 5000)
